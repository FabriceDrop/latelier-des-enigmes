L’Atelier des Énigmes – TikTok App

L’Atelier des Énigmes is a content creation application designed to generate and publish short puzzle and riddle videos on TikTok.

App Functionality

The app automatically creates short videos featuring a riddle, optional hints, and a final answer. These videos are intended for entertainment and educational purposes.

Once a video is generated, the app uses TikTok APIs to authenticate the account owner and publish the video directly to the connected TikTok account.

TikTok API Usage

The application uses TikTok APIs only for:

  • User authentication via TikTok Login Kit
  • Publishing videos to the authenticated TikTok account

Data Usage and Privacy

The app does not collect, store, process, or analyze personal user data. No user comments, messages, followers, or private information are accessed.

All actions performed by the app are initiated by the account owner for their own TikTok account.

Purpose of This Website

This page provides an overview of the application for TikTok App Review and compliance purposes.
